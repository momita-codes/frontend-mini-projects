## Employee Management System

An interactive mini-project demonstrating Employee Information using JavaScript array methods and DOM manipulation.

## Features

- Display all employees — View complete employee records
- Calculate total salaries — Get payroll total with reduce()
- Filter by department — View HR employees with filter()
- Search by ID — Find specific employee with find()

## Tech Stack

- HTML5
- CSS3
- JavaScript

## JavaScript Concepts Demonstrated

- `Array.map()` — Transform employee data to HTML
- `Array.filter()` — Filter employees by department
- `Array.find()` — Search employee by ID
- `Array.reduce()` — Calculate total salaries
- DOM manipulation with `innerHTML`
- Event handling with `onclick`

## How to Use

1. Open `employee_details.html` in your browser
2. Click buttons to:
   - Display all employees
   - Calculate total payroll
   - View HR department employees
   - Search employee by ID

## Files

- `employee_details.html` — Main interface
- `employee_details.js` — Business logic and array operations
